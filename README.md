# 💌 情书 Love Letter - 在线卡牌游戏

经典桌游《情书》的在线版，拥有高级AI对战系统。

## 🎮 功能特性

- 完整的《情书》2人对战规则
- 🧠 高级AI：记牌算率 · 牧师情报利用 · 伯爵夫人诈唬 · 分阶段战术
- 📊 分步骤展示AI行动过程，清晰易懂
- 📱 响应式设计，手机电脑均可游玩

## 🚀 部署方式

### 方式一：Vercel（推荐，最简单）

1. 在 [GitHub](https://github.com) 创建一个新仓库
2. 将本项目文件全部上传到该仓库
3. 前往 [vercel.com](https://vercel.com)，用 GitHub 账号登录
4. 点击 "New Project" → 选择你的仓库 → 点击 "Deploy"
5. 等待约1分钟，部署完成后你会获得一个 `xxx.vercel.app` 的链接

### 方式二：Netlify

1. 前往 [netlify.com](https://netlify.com)
2. 将整个项目文件夹拖拽到 Netlify 的部署区域
3. 或连接 GitHub 仓库自动部署

### 方式三：本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

构建后的文件在 `dist/` 目录中，可以部署到任意静态文件托管服务。
